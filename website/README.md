# [wikiplaylist.com](https://wikiplaylist.com)

### Development

Install dependencies:

```bash
yarn
# or
npm install
```

Running locally:

```bash
yarn dev
# or
npm run dev
```

### Tailwind CSS config

Use the following command when you add a tailwind plugin that adds to tailwind's base css:

```bash
yarn run build:base-css
# or
npm run build:base-css
```
